# deno-giphy

deno-giphy

## Requirement:
 - install deno (0.42.0)

## Testing and run:
```
// install deps
$ deno cache server.ts

// run in local
$ deno --allow-net server.ts

// allow all permission
$ deno -A server.ts

// run test case
$ deno test

// format code
$ deno fmt
```
